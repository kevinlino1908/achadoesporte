# Achado Esporte - Landing Page

Este repositório publica automaticamente a landing page estática no **GitHub Pages**.

## Como publicar no GitHub Pages

1. Faça push para a branch principal do repositório (`main`, `master` ou `work`).
2. Vá em **Settings > Pages** no GitHub.
3. Em **Build and deployment**, selecione **Source: GitHub Actions**.
4. Aguarde o workflow **Deploy static content to Pages** finalizar.

Após o deploy, a landing estará disponível na URL do Pages do repositório.
